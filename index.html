Definición

Css ('Cascading StyleSheets') es un lenguaje de formateo de código de lenguajes de marcas por lo que se dice que es un lenguaje complementario. Css está diseñado para formatear el texto añadiendo separaciones, colores, fuentes, capas o layout, etc para mejorar la interacción y visualización de texto o elementos multimedia insertados, por ejemplo, en un texto HTML
